Racquel: An ORM for Racket

Racqual is an object relational modeller for Racket. It consists of several components.

  • An extension of Racket’s class system that allows mapping of database tables to classes.

  • A generator that automatically creates mapped classes using database schema meta-data.

  • Functions for saving and deleting persistent objects.

  • RQL: An S-expression based SQL-like query language.

  • Serializers for serializing objects to and from JSON or XML.
